# Configuração do Feed do Instagram — Site de Imóveis

Guia rápido para exibir automaticamente as postagens do Instagram no site.

---

## 🎯 Objetivo

Exibir o feed do Instagram (conectado à Página de Imóveis) diretamente no site, com atualização automática a cada nova postagem.

---

## 🔌 Opção 1 — WordPress (via plugin)

Se o site é em WordPress, use um plugin pronto:

- **Smash Balloon Social Photo Feed** — mais popular, versão gratuita funcional
- **Spotlight Social Media Feeds** — interface moderna, boa personalização

### Passos

1. Acessar **Painel WordPress → Plugins → Adicionar Novo**
2. Buscar pelo nome do plugin
3. Instalar e ativar
4. Conectar a conta do Instagram (login via Facebook Business)
5. Configurar layout (grid, carrossel, colunas)
6. Copiar o shortcode gerado (ex: `[instagram-feed]`)
7. Colar na página de imóveis onde o feed deve aparecer

---

## 🌐 Opção 2 — Site em HTML (via widget)

Se o site é em HTML puro, use uma ferramenta de widget:

- **Elfsight Instagram Feed** — mais completo, freemium
- **LightWidget** — simples e gratuito

### Passos

1. Criar conta na plataforma escolhida
2. Conectar a conta do Instagram
3. Personalizar o layout (cores, quantidade de posts, formato)
4. Gerar o código de incorporação (embed)
5. Colar o código no HTML da página de imóveis, onde o feed deve aparecer

---

## ✅ Checklist

- [ ] Conta do Instagram é do tipo **Business** ou **Creator**
- [ ] Instagram conectado à Página do Facebook
- [ ] Plugin/widget instalado
- [ ] Conta autenticada com sucesso
- [ ] Feed exibindo na página de imóveis
- [ ] Teste de atualização: nova postagem aparece automaticamente

---

## 💡 Recomendação

Para o site de imóveis, a **Opção 1 (WordPress + Smash Balloon)** é a mais indicada por:

- Atualização automática sem manutenção
- Configuração visual (sem código)
- Suporte a cache e performance
